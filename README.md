# My Todo List App
나만의 투두리스트를 이용해 스케줄을 관리할 수 있습니다.

## Install
```
$ npm install
```

## Usage
- client
```
$ npm run start
```
- server
```
$ cd data
$ json-server --watch data.json --port 3001
```
